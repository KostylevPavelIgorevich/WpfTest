Для размещения тестов я сделал выбор в пользу JSON.
Для того чтобы использовать функционал, необходимо запустить программу и в окне выбрать пункт "Создать тест", затем задать название теста, например "C# базовый уровень".
Далее, в графе "Текст вопроса" описываем вопрос, например: "Что такое ООП?"
Затем нажимаем кнопку "Добавить ответ" — в появившемся поле пишем один из вариантов ответа. Чтобы добавить следующий вариант ответа, также нажимаем "Добавить ответ".
Таким образом можно задать несколько вариантов ответа.
После того как закончили с вариантами ответа, указываем порядковый номер правильного варианта в графе "Правильный ответ (номер)", затем нажимаем кнопку "Добавить вопрос".
Вопрос сохраняется, и теперь можно повторить процедуру столько раз, сколько нужно, чтобы создать нужное количество вопросов.
После того как закончили добавлять вопросы, нажимаем "Сохранить тест". После уведомления о том, что тест успешно сохранён, можно нажимать кнопку "Пройти тест". Далее, в открывшемся окне выбираем нужный нам тест и открываем его.
Откроется интерактивное меню, где будет отображён вопрос и варианты ответа на него. Отвечаем на все вопросы (нажимаем на правельный вариант ответа), и в конце появится окно с результатами теста.
